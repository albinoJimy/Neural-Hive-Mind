Guard Agents foi instalado com sucesso!

1. Verifique o status dos pods:
   kubectl get pods -l app.kubernetes.io/name=guard-agents -n {{ .Release.Namespace }}

2. Visualize os logs:
   kubectl logs -l app.kubernetes.io/name=guard-agents -n {{ .Release.Namespace }} -f

3. Acesse as métricas Prometheus:
   kubectl port-forward svc/{{ include "guard-agents.fullname" . }} 9090:{{ .Values.service.ports.metrics }} -n {{ .Release.Namespace }}
   # Acesse http://localhost:9090/metrics

4. Verifique o health check HTTP:
   kubectl port-forward svc/{{ include "guard-agents.fullname" . }} 8080:{{ .Values.service.ports.http }} -n {{ .Release.Namespace }}
   # Acesse http://localhost:8080/health/readiness

5. Verifique o health check gRPC:
   kubectl port-forward svc/{{ include "guard-agents.fullname" . }} 50051:{{ .Values.service.ports.grpc }} -n {{ .Release.Namespace }}

6. Endpoints da API:
   # Listar incidentes
   curl http://localhost:8080/api/v1/incidents

   # Listar ações de remediação
   curl http://localhost:8080/api/v1/remediation

   # Criar incidente (teste)
   curl -X POST http://localhost:8080/api/v1/incidents -H "Content-Type: application/json" -d '{...}'
