1. Verifique o status dos pods:
   kubectl get pods -l app.kubernetes.io/name=scout-agents -n {{ .Release.Namespace }}

2. Visualize os logs:
   kubectl logs -l app.kubernetes.io/name=scout-agents -n {{ .Release.Namespace }} -f

3. Acesse as métricas Prometheus:
   kubectl port-forward svc/{{ include "scout-agents.fullname" . }} 9090:9090 -n {{ .Release.Namespace }}
   # Acesse http://localhost:9090/metrics

4. Verifique o health check:
   kubectl port-forward svc/{{ include "scout-agents.fullname" . }} 8000:8000 -n {{ .Release.Namespace }}
   # Acesse http://localhost:8000/health/ready

5. Veja sinais detectados (API):
   curl http://localhost:8000/api/v1/signals

6. Simule um sinal (para testes):
   curl -X POST http://localhost:8000/api/v1/signals/simulate?domain=TECHNICAL
